# 구현 계획서 (Implementation Plan)

## 목표 설명
도널드 트럼프, 이재명, 일론 머스크의 최근 일주일간 SNS 활동을 보여주는 웹 페이지를 제작합니다. HTML5, CSS, Javascript를 사용하며, 시각적으로 매력적이고 직관적인 UI를 목표로 합니다.

## 사용자 리뷰 필요 사항
- 실제 API 연동은 클라이언트 사이드(HTML/JS) 만으로는 CORS 이슈 및 인증 키 문제로 제한될 수 있습니다. 본 프로젝트에서는 **Mock Data(더미 데이터)** 구조를 통해 실제 API 응답과 유사한 형태로 구현하고, 추후 백엔드나 API 키가 준비되었을 때 쉽게 교체할 수 있도록 설계합니다.
- 각 인물의 대표 색상 및 스타일 테마에 대한 확인이 필요할 수 있습니다. (예: 트럼프-빨강/네이비, 이재명-파랑, 일론머스크-검정/X로고)

## 변경 제안

### 기본 구조
- **HTML**: 시맨틱 태그를 활용한 구조화 (`header`, `main`, `section`, `article`).
- **CSS**: 바닐라 CSS를 사용하되, CSS 변수(Root variables)를 활용하여 테마 관리. Flexbox 및 Grid 레이아웃 활용.
- **JS**: ES6+ 문법 사용. 모듈화를 통해 데이터 처리와 DOM 조작 분리.

### 주요 컴포넌트
#### `index.html`
- 메인 페이지 구조.
- 인물별 섹션 또는 통합 타임라인 뷰 제공.

#### `css/style.css`
- 전체적인 스타일링.
- 인물별 카드 디자인 (Card UI).
- 반응형 미디어 쿼리.

#### `js/api.js` (Simulated Real API)
- `fetchSNSData()`: 실제 인물들의 최근 발언(2025~2026년)을 담은 고정 데이터셋 반환.
- 데이터의 날짜는 함수 호출 시점에 맞춰 최근 7일 이내로 재가공하여 반환 (UI 필터링 테스트용).

#### `js/app.js`
- 초기화 및 렌더링 로직.
- 이벤트 리스너 (클릭 이벤트 등).

## 검증 계획

### 자동화된 테스트
- 별도의 테스트 프레임워크 없이, 브라우저 콘솔을 통한 데이터 로딩 확인.

### 수동 검증
1. **새로고침 테스트**: 페이지 새로고침 시 로딩 애니메이션 후 데이터가 표시되는지 확인.
2. **기간 필터링**: 표시된 데이터가 현재 시점으로부터 7일 이내인지 확인.
3. **링크 이동**: 카드 클릭 시 해당 실제 SNS 게시글로 이동하는지 확인.
4. **UI 반응성**: 모바일 및 데스크탑 해상도에서 레이아웃이 깨지지 않는지 확인.
